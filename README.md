# CRKBD Keymaps

Personal keymap exports and layer references for Corne/crkbd keyboards.

This repository keeps each keyboard revision in its own directory, with the editable keymap export and rendered layer images checked in together.

## Keymaps

| Directory | Keyboard | Config file | Tool |
| --- | --- | --- | --- |
| [`corne-v3`](corne-v3/) | Corne v3 / Helidox | [`crkbd.layout.json`](corne-v3/crkbd.layout.json) | [VIA](https://www.usevia.app/) |
| [`corne-v4`](corne-v4/) | Corne v4 | [`corne.vil`](corne-v4/corne.vil) | [Vial](https://vial.rocks/) |

Each directory includes its own README with screenshots for layers 0-3.

## Layout Notes

- Layer 0 is the base typing layer.
- Layer 1 contains numbers, arrows, and macros.
- Layer 2 contains symbols and shifted symbols.
- Layer 3 contains keyboard controls, RGB controls, and media keys.

## Updating

1. Open the relevant keymap file in VIA or Vial.
2. Make the layout changes.
3. Export the updated keymap back into the matching directory.
4. Update the layer screenshots if the visual layout changed.
